## 🎯 How It Works

1. The target receives a link to a mysterious "pending notification."
2. The target clicks the glowing gift box.
3. **Domi** (the Chief Enforcer Doberman) appears, completely unamused by their life choices.
4. A high-velocity virtual slipper is launched directly into the camera lens.
5. The successful hit is permanently logged in a global Firebase database for ultimate bragging rights.

## ✨ Features

* **Sleek UI:** Dark mode by default, because we are developers.
* **Advanced Physics Engine:** State-of-the-art CSS keyframe animations for maximum slipper impact.
* **Global Telemetry:** Real-time Firebase Realtime Database integration tracks Links Sent, Boxes Opened, and Confirmed Slipper Shots across the globe.
* **Completely Serverless:** Hosted entirely on GitHub Pages. No backend maintenance required.

## 🚀 Deployment 

If you want to host your own instance of the Digital Serupala Adipen:

1. Fork or clone this repository.
2. Create a free [Firebase](https://firebase.google.com/) Realtime Database (set it to Test Mode).
3. Replace the `firebaseConfig` object in `script.js` with your own keys.
4. Deploy the `index.html` file using GitHub Pages.
5. Send the live link to the person who just broke your build.
